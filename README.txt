OBJECTIFS :

    Le travail est à réaliser en langage Java.
    Le travail est à réaliser en binôme.
    La constitution en binômes sera réalisée par les enseignants de TP.

    On s'attachera à structurer les données manipulées en types agrégés.
    On s'attachera à résoudre les problèmes rencontrés par la programmation de fonctions ad-hoc.
    La fonction principale pourra être architecturée autour d'un système de gestion de menus textuels permettant son utilisation.
    Tous les types agrégés programmés seront documentés sous la forme d'un commentaire placé dans le code source avant le type agrégé et incluant la description du rôle du type et la description de ses champs.
    Toutes les fonctions programmées seront documentées sous la forme d'un descriptif placé dans le code source avant la fonction et d'éventuelles lignes de commentaires inline.

    Le fichier java développé devra inclure une première ligne de commentaires avec les noms et prénoms des développeurs, le groupe de TP auquel ils appartiennent et le numéro de projet choisi.
    Le fichier .java devra être accompagné d'un fichier readme.txt (ou .pdf ou .doc ou .ood) incluant:
        le mode d'emploi de l'application développée,
        un descriptif de la technique employée pour résoudre chaque problème,
        l'indication de ce qui a été développé par chacun des 2 membres du binôme,
        toutes autres informations jugées pertinentes.
    Chaque binôme réalisera une présentation (environ 10 mn) du fonctionnement de son application (assortie éventuellement de questions de l'enseignant de TP) au cours d'une séance de TP de la semaine du 11 mai 2015.
    Les projets sont à rendre pour le jeudi 14 mai 2015 à 14h dernier délai.
    Les deux fichiers sont à envoyer par courrier électronique à l'adresse nicolas.janey@univ-fcomte.fr. L'objet du mail devra contenir les noms des auteurs, leur numéro de groupe de TP et le numéro de projet choisi. Un accusé de réception sera émis à la réception du message.
    Tout projet non rendu ou rendu en retard sera noté à 0.0.
